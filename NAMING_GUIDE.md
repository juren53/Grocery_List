# Quick Reference Grocery Naming Guide

This guide helps you format grocery items for optimal categorization by the Grocery List Organizer.

## General Naming Principles

### âœ… DO: Use Common, Generic Names
- **Be descriptive but simple** - Include key identifiers that match category keywords
- **Use natural language** - Write items as you would say them
- **Include important modifiers** - Brands, types, or varieties that help with matching

### âŒ DON'T: Overthink It
- **Don't worry about exact formatting** - The system is flexible

## Quick Categorization Test

Not sure if your item name is good? Ask yourself:

1. **Would someone else know what I'm looking for?**
2. **Does it include the main product type?** (bread, milk, tea, etc.)
3. **Does it include important variety info?** (green tea, whole wheat bread, etc.)
4. **Could this item be in multiple store sections?** (like corn or tea?)

If yes to all, you're good to go! âœ…

## What If It's Still Uncategorized?

If an item doesn't get sorted automatically:

1. The program will ask you to categorize it interactively
2. You can add it as a new keyword so it sorts correctly next time
3. The system learns from your choices and gets better over time

**Pro tip:** The more specific you are with item names when adding keywords, the better future categorization will be!
